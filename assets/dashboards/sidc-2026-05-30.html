  • 예정(8월)은 잠정 projection이다. 2026-05-30 총액·비중에 Architect 3건 매매를 적용한 값으로, 월 적립(₩800K/월)·가격 변동·분배금 누적은 미반영. 8월 하순 실제 집행 시 그때의 holdings로 Architect를 재호출해 재산출해야 한다.
  • spec 목표는 종착점이지 8월 목표가 아니다. 8월은 4분기 front-loaded 전환의 2차 분할이며, Core 51%·예수금 15%·Satellite 34%로의 완전 수렴은 11월(4차) 이후로 이연된다.
  • 미분류(금, TIGER KRX금현물 1.18%)는 spec 4개 카테고리에 매핑되지 않은 상태. satellite_additional 편입 / 카테고리 신설 / 매도 중 택1이 미결로, 매매가 아닌 분류 결정 사안이다.
  • 이 대시보드는 시각화 보조 도구다. 새로운 권고를 생성하지 않으며, 기존 SIDC 산출물(holdings.yaml · Architect 2026-05-30)을 렌더링할 뿐이다. 매매·결정은 진성이 의사결정 로그와 함께 직접 판단한다.
